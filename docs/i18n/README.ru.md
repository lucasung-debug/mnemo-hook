# mnemo-hook

**Слой персистентной памяти для Claude Code** — ваш ИИ помнит, что происходило в прошлой сессии.

> *mnemo* (греч.: mneme, память) — потому что каждая сессия должна начинаться там, где закончилась предыдущая.

[English](../../README.md) | [Korean](README.ko.md) | [Chinese](README.zh.md)

---

## Проблема

Каждый раз, когда вы начинаете новую сессию Claude Code, контекст теряется. Вам приходится заново объяснять, над чем вы работали, какие решения были приняты, что ещё не завершено. **mnemo-hook** решает эту проблему, автоматически сохраняя и восстанавливая историю сессий через систему хуков Claude Code.

## Возможности

- **Брифинг сессии** — При старте новой сессии показывает итоги предыдущей: решения, прогресс, открытые задачи
- **Авто-сохранение** — Автоматически фиксирует git-коммиты, вехи записи файлов и ключевые решения
- **Тихие подсказки** — Во время работы показывает релевантные воспоминания из прошлых сессий
- **Глубокий поиск** — Введите `recall {тема}` для полнотекстового поиска по всей памяти
- **Граф знаний** — Строит граф проекта + визуализация Obsidian Canvas (опционально)
- **Приватность** — Автоматически удаляет пароли, токены, приватные ключи
- **Безопасность** — Защита от path traversal, ReDoS, YAML-инъекций. Автоматическое обнаружение токенов GitHub/AWS/Anthropic/Slack/JWT

## Быстрый старт

```bash
git clone https://github.com/lucasung-debug/mnemo-hook.git
cd mnemo-hook
npm test
npm run install-hooks
# Перезапустите Claude Code
```

## Требования

- Node.js >= 18
- Claude Code
- Нулевые зависимости npm
