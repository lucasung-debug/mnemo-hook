mnemo-hook

A persistent, local memory layer for Claude Code using hooks, Markdown storage, privacy filtering, recall, and optional Obsidian graph output.

Node.js >= 18 Zero npm dependencies Local file storage MIT licensed

What It Solves

Claude Code sessions can lose useful project history between invocations. mnemo-hook saves sanitized local memory so later sessions can recover decisions, milestones, open work, and related context.

Session continuity

Briefings, milestones, decision capture, quiet hints, and explicit recall {topic} search.

Privacy first

Redacts credentials and sensitive metadata before memory is written to disk.

Inspectable storage

Memories are local Markdown/YAML files, with optional Obsidian graph and Canvas output.

Install And Validate

git clone https://github.com/lucasung-debug/mnemo-hook.git
cd mnemo-hook
npm run check
npm test
npm run install-hooks

Note: mnemo-hook is an open-source CLI/hook utility, not a hosted web SaaS.
